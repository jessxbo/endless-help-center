Before you start

A few things to know before downloading.

Why do I need Godot?

Godot is the free, open-source game engine we use across our programs. You'll need it for Explore: Godot, Core: Threadbare, and More: Threadbare, where you'll build levels, characters, mechanics, and full StoryQuests.

Threadbare itself is built in Godot, so installing it locally is what lets you contribute to the game.

Which version of Godot do I need?

Install the most recent stable version of Godot 4 from the official Godot download page. Avoid Godot 3.x, which is an older version with different syntax and project formats.

You don't need the .NET / C# version unless your program specifically tells you to. The standard build is what we use.

What are the system requirements?

Godot is lightweight and runs on most modern computers. The basics:

  • Operating system: Windows, macOS, or Linux
  • RAM: 8 GB minimum for Godot-based programs
  • Internet: Reliable connection for downloads, GitHub, and the Endless Studios platform

For more on tech requirements, see Troubleshooting → System requirements.

Installing Godot

Good news: Godot is self-contained and doesn't require a traditional install. Download, extract, and run.

How do I install Godot on Windows?
  1. Go to godotengine.org/download/windows
  2. Click Download Godot Engine (the Standard version, not .NET). For most computers, choose Windows - x86_64.
  3. Extract the downloaded .zip file to a folder of your choice.
  4. Double-click the Godot_v4.x.x-stable_win64.exe file to launch Godot.

That's it. Godot doesn't need a traditional installation, just extract and run.

How do I install Godot on macOS?
  1. Go to godotengine.org/download/macos
  2. Click Download Godot Engine (the Standard version, not .NET). The Universal build works on both Apple Silicon and Intel Macs.
  3. Extract the downloaded .zip file.
  4. Drag the Godot app into your Applications folder, then open it from there.

Godot for macOS is code-signed and notarized, so you shouldn't get a security warning. If you do, right-click the app and choose Open.

How do I install Godot on Linux?
  1. Go to godotengine.org/download/linux
  2. Click Download Godot Engine (the Standard version, not .NET). Most users want Linux - x86_64.
  3. Extract the downloaded .zip file.
  4. Make the binary executable, then run it: chmod +x Godot_v4.x.x-stable_linux.x86_64 followed by ./Godot_v4.x.x-stable_linux.x86_64

You can also install Godot from Flathub if your distribution supports Flatpak.

How do I know if Godot is installed correctly?

When you launch Godot, you should see the Project Manager window appear. This is where you'll create new projects or open existing ones.

At the bottom of the Project Manager window, you should see the version number. Make sure it starts with 4. (for example, 4.7-stable or 4.6.3-stable). If you see a version starting with 3., you have the older Godot 3.x, which won't work for our programs.

If Godot won't launch or you see an error, check the official Godot troubleshooting page for common issues.
